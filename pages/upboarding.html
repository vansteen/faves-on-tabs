Release Notes

v1.2

  • Bookmark groups — bookmarks in the configured folder are now organized into groups when subfolders are present. Each subfolder renders as a titled section on the new tab page. Direct bookmarks not in any subfolder are collected into an "Others" group at the end. When no subfolders exist, the flat list layout is unchanged.
  • Font size controls — adjust the font size directly on the new tab page using the A− and A+ buttons in the header. The preference is saved and applied immediately. The same setting is also available in the preferences page via a slider (70%–150%).
  • Various small bug fixes and visual improvements.
